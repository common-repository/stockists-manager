Stockists Manager - show product's stockists on map or list based on user location. Stockist info page with street view map available

== Description ==

Stockists Manager is a WordPress plugin that add possibility to manage stockists and show them on the product page(WooCommerce) ordered by location closest to the user. 

= Features: =
* Display options: list or map.
* Custom pins for map
* Stockist info page: logo, description, gallery, etc
* Map type, zoom, distance, amount to display per page, default location and other options can be managed through settings page.
* Categories. One stockist can be added to several categories
* Premium stockists




= Next Release ( 1.1.0 ) =

= Features: =
* "No stockists" text editable in admin
* Stockists search page template. Admin can create a page and select as a template Stockists Search Page. There will be
 postcode search option, limit by radius, keyword, category.
* "Premium Marker icon URL" for categories.
* CSS class for premium stockists. This will provide option to add styles for Premium Stockists
* Adding filters and actions for developers

= Changes: =
* stockists will not be shown if Stockists options for product is not chosen and Stockist is not enabled by default for all products in settings

== Installation ==

1. Install Stockists Manager either via the WordPress.org plugin directory, or by uploading the files to your server
2. Activating Stockists Manager and you're ready to go!
